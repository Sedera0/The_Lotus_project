# Le Lotus Bungalow

## Sections créées par Sedera :
1. **Header (En-tête)**:
   - Navigation (avec liens vers les différentes sections)
   - Hero Section (Section héros)
2. **Plats Section (Section Plats)**:
   - Présentation des plats et des spécialités culinaires
3. **Reservation Section (Section Réservation)**:
   - Formulaire de réservation
4. **Footer (Pied de page)**:
   - Informations de contact
   - Liens vers les réseaux sociaux
   - À propos du service

## Sections créées par Mandresy :
1. **Bungalow Section (Section Bungalow)**:
   - Présentation des différents types de chambres
2. **About Us Section (Section À propos de nous)**:
   - Description de l'établissement et de ses services


## Frameworks utilisés :
- **Flickity**: Un framework JavaScript pour créer des carrousels et des galeries d'images.
- **Font Awesome**: Une bibliothèque d'icônes vectorielles et d'icônes sociales utilisées pour les icônes dans la navigation et le formulaire de réservation.
- **Google Fonts**: Utilisé pour importer des polices personnalisées (`Montserrat Alternates` et `Dancing Script`) pour le texte du site.
